# Glossary

## AGENTS

Repo-owned instructions for coding agents. This is a canonical behavioral and architecture rule file, not a synthesized summary.

## AI Context Pack

The files under `krukraft-ai-contexts/`. These are concise, maintained summaries of current project truth and operational expectations.

## Raw Source

A document, note, export, or snapshot that should be treated as evidence rather than rewritten truth.

## Wiki Page

A synthesized markdown page under `knowledge/wiki/` that organizes repo knowledge and links back to raw or canonical sources.

## Canonical Source

The highest-priority source when two documents disagree. For this repo that usually means code/runtime, `AGENTS.md`, or `krukraft-ai-contexts/`.

## Inference

A conclusion derived from multiple sources. Inferences are allowed in wiki pages, but they must not be written as direct facts without support.

## Lint

A structural and consistency pass over the knowledge layer. For phase 1 it checks file presence, required headings, and index coverage.
