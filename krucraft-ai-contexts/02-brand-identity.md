## Brand Name Decision

- **KruCraft** was chosen over alternatives like PaperDock, StudyDock, LessonMarket, etc.
- The name was rated **8.5/10** overall (9/10 memorability, 9/10 Thai market clarity, 8/10 branding, 7/10 global reach)
- Final production domain: **krucrafts.com** (with 's' since krucraft.com was unavailable)

- Modern SaaS startup
- Creator economy (Gumroad / Notion / Canva vibes)
- Trustworthy but creative
- Platform for Thai teachers and educational creators

## Logo Design Spec

### Core Concept

- **KRU** = strong, authoritative, platform identity (teachers / creators)
- **Craft** = creative, human, expressive layer (marketplace / creation)

### Layout

- Square app-icon style container (rounded corners)
- Centered composition
- Two-line structure: KRU (primary, top) / Craft (secondary, bottom)

### Typography

- **KRU:** Bold geometric sans-serif, strong, stable, slightly condensed, high readability at small sizes
- **Craft:** Custom script or soft rounded font (base: Pacifico or Lobster family, then refined)
  - Friendly, creative, slightly playful
  - Must remain readable at small sizes

### Visual Balance

- Reduce dominance of "KRU" slightly
- Increase clarity and legibility of "Craft"
- Balanced spacing and alignment

### Background

- Deep blue / indigo base: `#2D2F8F` or similar
- Subtle grid system: 5-8% opacity, very low contrast, not distracting

### Style

- Flat + soft modern UI
- No heavy gradients
- Optional very subtle lighting or depth (minimal)
- Clean edges, vector-friendly

### Deliverables

1. Primary logo (1:1 square format, 4K)
2. Simplified icon version (for favicon/app icon)
3. Monochrome version (black/white)
4. Transparent background version + solid background version

### Font Analysis (Craft word)

The "Craft" script is custom lettering based on Pacifico/Lobster family with:
- Stroke thickness semi-consistent (not pure brush)
- Terminal curves rounded → friendly SaaS vibe
- "f" and "t" simplified for readability
- Kerning adjusted for logo use

**Closest font matches:** Pacifico (top pick), Lobster Two, Satisfy, Balqis
