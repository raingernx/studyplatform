# PaperDock Launch Checklist

## Marketplace

- Resource library loads correctly
- Resource detail pages display correctly
- Resource search and filters work

## Upload System

- Admin can upload a resource file
- Preview images upload successfully
- Version history works

## Payments

- Stripe checkout works
- Purchase record is created in database
- Stripe webhook is verified

## Downloads

- Download endpoint checks purchase authorization
- Files download correctly
- Download count increments

## Admin Tools

- Admin can create/edit resources
- Categories and tags work
- Resource status (draft/published) works

## Security

- Protected API routes require authentication
- Downloads require valid purchase
- Admin routes require ADMIN role
